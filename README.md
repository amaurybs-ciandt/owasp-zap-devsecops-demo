# OWASP ZAP DevSecOps Demo

This repository demonstrates the integration of OWASP ZAP (Zed Attack Proxy) into a DevSecOps workflow. It provides sample configurations and automation scripts to help you incorporate security testing into your CI/CD pipelines.

## Features
- Example GitHub Actions workflow for automated security scanning
- Sample Terraform configurations for secure infrastructure
- Documentation for setting up and running OWASP ZAP scans

## Getting Started
1. Clone this repository.
2. Review the provided GitHub Actions workflows in `.github/workflows/`.
3. Customize the Terraform files for your environment.
4. Integrate OWASP ZAP scans into your CI/CD pipeline.

## Repository Structure
- `azure-aks-cluster-sec-lab/`: Secure AKS cluster Terraform lab
- `azure-aks-cluster-security-terraform/`: Additional Terraform security examples
- `owasp-zap-devsecops-demo/`: OWASP ZAP integration and documentation

## Requirements
- Terraform
- GitHub Actions
- OWASP ZAP
